Help

For details on how this works, the best choice is looking at the source code: Sony TV Web Page Remote source code.

Using this requires the TV to support IRCC control (Sony Bravia TV infrared like control commands), have a fixed IP address, and allow remote access using a pre-shared key.

Set pre-shared key on your TV by accessing this or similar page on your TV: [Settings] [Network] [Home Network Setup] [IP Control] [Pre-Shared Key]
Set [Authentication] to [Normal and Pre-Shared Key]
Enter your selected key there, and on this page in the Setup form.

If the TV IP address is not fixed, give your TV a static IP address or make a DHCP reservation for a specific IP address in your router.

Enter the TV IP and Key in the Setup form on this page, and click Save.
Once that is done, the buttons on the page can be used to send commands.

Enter the list of over-the-air channels you want direct access to.
This will create a button for each channel.

The Javascript code prints out status and error messages to the Javascript Console.
To see the messages, open up your browser's Developer Tools or similar to see the Console messages.
For example, in Firefox type Control-Shift-J or click on [Menu] [Developer] [Browser console] to open the Console and look at the JS (Javascript) Tab to see messages from this script.

Code Changes

The extension as shipped should be usable by most everyone, since it also allows for custom set of channel buttons.
But if additional buttons are needed, the code can be edited locally.
The web page can be edited as needed by downloading the entire web extension source code locally.
The extension can then be loaded from the local directory.
When making such edits, make sure to enter the TV IP and Key into the variables otherwise you will have to enter setup each time you restart your browser, since the extension has to reloaded on each restart.
To locally load the add-on in Firefox: visit the about:debugging page and then in [Add-ons] use the [Load Temporary Add-on] button and select any file in the folder where this extension is stored.
In Chrome, visit [More Tools] [Extensions] and use the [Local unpacked extension...] command.
(This has only been tested on Firefox.)
